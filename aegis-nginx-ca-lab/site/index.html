Aegis Nginx Local CA Lab

This page is served by Nginx with a certificate issued by a lab-local root CA. If your Firefox build shows this page with a secure lock, the local CA is trusted and the Aegis bundle approved it for localhost.

Quick checks:

  • The certificate should chain to Aegis Nginx Lab Root CA.
  • With the default bundle, the connection should succeed.
  • With -OmitRootApproval, the same site should be blocked by Aegis.
